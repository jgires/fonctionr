# load_and_active_fonts

Function to load and activate fonctionr's built-in fonts. Available
fonts, included in the package itself, are `"Roboto"`, `"Montserrat"`,
`"Gotham Narrow"`, and `"Euclid Circular A"`. Default is `"Roboto"`.

## Usage

``` r
load_and_active_fonts()
```

## Value

No return value, called for side effects.

## Examples

``` r
# Loading of fonts from fonctionr. You can now use it in fonctionr !
load_and_active_fonts()
```
